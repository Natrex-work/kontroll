# Endringer V1.5 - kart, OCR og avvik

- Oppdatert appversjon til `1.5.0` / `V1.5` og bumpet cache/service worker.
- Kart og Område viser lovregulerte verne-/forbudslag i kollapsbart temakart, med kystnære fiskeridata filtrert bort.
- Temalag og områdekort er gjort responsive på mobil.
- Standardtekstpolering fjerner `ved kontrollposisjonen` og bruker nærmeste sted/kontrollsted.
- OCR er styrket med større bildegrunnlag, flere OCR-varianter, deskew som standard og adaptiv terskling når OpenCV er tilgjengelig.
- Flere lenker per avvik vises nå som faner: Lenke 1, Lenke 2, Lenke 3 osv.
- Avvik fra flere områdetreff beholdes som egne automatiske avvik.
