# 1.8.1 - Beslag/avvik klikkretting

Rettet konkret feil der Legg til redskap/beslag/Avvik + beslag ikke var tydelig synlig eller robust klikkbar på iPhone.

Endringer:
- Lagt til synlig knapp i selve kontrollpunktkroppen, rett under notatfeltet.
- Topphandling skjules på smal skjerm for å unngå at den forsvinner bak/havner utenfor kortet.
- Klikkhåndterer bruker nå nærmeste faktiske knapp/action-element.
- Klikk setter punktet til Avvik, oppretter avviks-/beslagsrad, genererer beslagsnummer, velger raden for bildebevis, rendrer UI og autosparer.
- Versjon/cache bumpet til 1.8.1.
