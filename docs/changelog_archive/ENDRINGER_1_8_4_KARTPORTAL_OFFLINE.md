# 1.8.4 - Kartportal og offline-reguleringer

- Kart og Område bruker Fiskeridirektoratets Experience Builder-kartportal som hovedvisning.
- Appens egne kartfunksjoner beholdes som rask punktkontroll ved posisjon.
- Kontrollkartet filtrerer etter valgt kontrolltype, art/fiskeri og redskap når slike valg finnes.
- Relevante reguleringsområder lagres lokalt som vektorobjekter uten automatisk nedlasting av kartfliser.
- Kystnære fiskeridata holdes ute av lov-/reguleringsfilteret.
- CSP er åpnet for portal.fiskeridir.no som iframe-kilde.
- Versjon og cache-bust er bumpet til 1.8.4.
