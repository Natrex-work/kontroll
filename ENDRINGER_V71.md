KV Kontroll v71
=================

Denne pakken samler de siste forbedringene i én full oppdatert versjon:

- Portal-lik kartvisning med leaf-lag og raster fra fiskeri- og vern-tjenestene
- Oppdatert kartcache-/lagvalgversjon (v71)
- Forbedret Person / fartøy med automatisk OCR, hint og registeroppslag
- Render-sikrere OCR-flyt uten tunge ekstra avhengigheter
- Kollapset OCR-tekstfelt i Person / fartøy

Bruk GitHub Desktop, behold .git, erstatt gamle prosjektfiler med innholdet i denne pakken, commit/push og kjør Clear build cache & deploy i Render.
