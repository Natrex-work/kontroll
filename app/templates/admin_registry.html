Last opp deltakerliste

Tableau-visningen til Fiskeridirektoratet blokkerer ofte automatisk nedlasting (krever pålogging eller godkjent klient). Last ned data manuelt og last opp her:

  1. Åpne tableau.fiskeridir.no — Påmeldte hummerfiskere
  2. Klikk på «Last ned» (Download) i Tableau-verktøylinjen
  3. Velg «Crosstab» eller «Data» som format
  4. Velg CSV som filtype og lagre filen
  5. Last opp filen her under
Hvilke kolonner gjenkjennes?
Parseren gjenkjenner kolonner med følgende navn (norske eller engelske, små eller store bokstaver):
  • navn / namn / fullt navn / name
  • deltakarnummer / deltakernummer / deltakarnr / participant_no
  • type fiskar / type fisker / fiskartype / fisher_type
  • sesong / år / year
