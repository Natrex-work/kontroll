# Innsendingscheckliste – Fiskerikontroll

- [ ] AppConfig.plist har riktig HTTPS-adresse
- [ ] Bundle ID og Team ID er satt
- [ ] Appnavn i App Store er satt til Fiskerikontroll
- [ ] Personvernerklæring er publisert på offentlig URL
- [ ] Skjermbilder for iPhone og iPad er tatt
- [ ] Demo-konto er opprettet til App Review
- [ ] Privacy-spørsmål i App Store Connect er fylt ut
- [ ] Export compliance er svart ut
- [ ] Archive er validert i Xcode Organizer
- [ ] Build er lastet opp til App Store Connect
